## 1. Overall Roadmap

```
W1: 将棋の世界へようこそ ← MVP
    盤面・駒紹介・歩・金・銀・飛・角
W2: 駒の動きをマスターしよう
    香・桂・王・全駒レビュー・駒を取る
W3: 取る・成る・打つ
    成りの基本・持ち駒・二歩・打ち歩詰め
W4: 王手と詰みを理解しよう
    王手・王手の対応・詰みの基本・詰み判定
W5: はじめての1手詰
    頭金・尻金/腹金・銀詰み・桂打ち・飛角詰み・ミックス
W6: もっと1手詰 & 3手詰の入口
    開き王手・両王手・逃げ道封鎖・3手詰の考え方
W7: 手筋を覚えよう
    歩の手筋・銀の手筋・金の手筋・飛角の手筋
W8: 囲いと戦法の入口
    矢倉・美濃・居飛車/振り飛車・棒銀・ミニマッチ
```

**設計思想**:
- Duolingo方式: ゲーム目的（詰み）をW4で導入（全体の前半）
- ただし将棋は駒種類が多いため、最低5駒（W1）+ 残り3駒（W2）→ ルール（W3）→ 詰み（W4）の順
- 1手詰をW5に配置。これがDuolingoでいう「チェックメイトパズル」に相当
- W7-8は応用。ここまで来た学習者は中級者への橋渡し段階

## 3. Concept Dependencies（概念の前提関係）

```
盤面認識 (1-1-1)
    └→ 駒の存在を知る (1-1-2)
        └→ ゲーム目的を知る (1-1-3)
            └→ 歩の動き (1-2-1)
                ├→ 歩で取る (1-2-2) ← "取る"概念の初出
                │   └→ 歩ドリル (1-2-3)
                └→ 金の動き (1-3-1) ← 歩との比較がある
                    ├→ 金で守る (1-3-2) ← "守り"概念の初出
                    │   └→ 金ドリル (1-3-3)
                    └→ 銀の動き (1-4-1) ← 金との比較が前提
                        ├→ 金銀比較 (1-4-2)
                        │   └→ 銀ドリル (1-4-3)
                        └→ 飛車 (1-5-1) ← "何マスでも"が新概念
                            └→ 角 (1-5-2)
                                └→ 飛角ドリル (1-5-3)
                                    └→ World 1 まとめ (1-6-1〜3)
```

**重要な概念初出マップ**:
| 概念 | 初出 | 補強 |
|------|------|------|
| 盤面（9x9） | 1-1-1 | 1-1-2 |
| 駒の向き（先手/後手） | 1-1-2 | — |
| ゲーム目的（王を詰ます） | 1-1-3 | — |
| 1マス移動 | 1-2-1 (歩) | 1-3-1 (金), 1-4-1 (銀) |
| 駒を取る | 1-2-2 | 1-3-2 |
| 方向数の違い | 1-3-1 (6方向) | 1-4-1 (5方向) |
| 守りの概念 | 1-3-2 | — |
| 攻めの概念 | 1-4-2 | — |
| 遠距離移動 | 1-5-1 (飛車) | 1-5-2 (角) |
| 駒を飛び越えられない | 1-5-1 | 1-5-2 |

## 4. Beginner Pitfalls（初心者つまずきポイント）

| # | つまずき | 該当Unit | 対策 |
|---|---------|---------|------|
| BP1 | 銀と金の動きの混同 | U1-4 | 金銀比較レッスン(1-4-2)で明示的に対比。「金=斜め後ろ以外」「銀=横と真後ろ以外」 |
| BP2 | 駒が多すぎて覚えられない | U1-1 | 「8種類だけど少しずつ」と安心させる。W1は5種類だけ |
| BP3 | 盤面の座標が分からない | U1-1 | タップで体験させる。座標ラベル表示 |
| BP4 | 飛車と角の混同 | U1-5 | 「飛車=タテヨコ、角=ナナメ」の対比。十字 vs X字 |
| BP5 | 「取った駒をどうするの？」 | U1-2 | 持ち駒の概念を軽く紹介（詳細はW3） |

## 5. "Too Hard Too Early" Risks

| リスク | 該当 | 判断 |
|--------|------|------|
| W1で成り(promotion)を教える | W1-U2-L3 (歩ドリル) で「成ると何に？→まだ習ってないよ！」 | OK: 伏線だけ。詳細はW3 |
| W1で持ち駒(drop)を教える | W1-U2-L2 で「取った駒は持ち駒に」と軽く触れる | OK: 概念紹介だけ。実際に打つのはW3 |
| W1で王手/詰みを教える | W1-U1-L3 で「王を詰ませる」をゲーム目的として紹介 | OK: 概念紹介だけ。実践はW4 |
| W1で香/桂を教える | しない | OK: W2に配置済み |
| W1で二歩を教える | しない | OK: W3に配置済み |
