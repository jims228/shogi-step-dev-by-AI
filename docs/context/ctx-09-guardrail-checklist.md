## G1: Duolingo に寄せすぎていないか

| チェック項目 | 現状 | Status |
|-------------|------|--------|
| アプリ名に"Duolingo"を含んでいないか | "Shogi Step" | OK |
| Duolingoの緑を使っていないか | 未決定（Q4: テーマカラー） | PENDING |
| フクロウやDuolingoキャラを模倣していないか | コーチキャラはオリジナル（Q1） | PENDING |
| ハート制をそのまま導入していないか | Q5: v1なし推奨 | PENDING |
| ロードマップUIがDuolingoのスクリーンショットのコピーではないか | 新デザイン予定 | PENDING |
| XP/ストリークの表示がDuolingo風すぎないか | v1は最小限のガミフィケーション | OK |

**判定基準**: 概念（bite-sized, roadmap, spaced review）は参考OK。視覚的デザイン・色・キャラ・レイアウトの直写しはNG。

## G3: レッスンが長くなりすぎていないか
